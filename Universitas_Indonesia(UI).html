    UNIVERSITAS INDONESIA

    Informasi Universitas

    • Lokasi: Depok, Jawa Barat, Indonesia
    • Link Resmi: https://www.ui.ac.id
    • Akreditasi Institusi: Unggul oleh BAN-PT

    Jalur Masuk

    1. SNBP: Jalur seleksi berdasarkan prestasi akademik.
    2. SNBT: Jalur seleksi berdasarkan tes berbasis komputer nasional.
    3. PPKB: Program Pemerataan Kesempatan Belajar untuk siswa dari daerah tertentu.
    4. SJP: Seleksi Jalur Prestasi non-akademik.
    5. Talent Scouting: Penjaringan bakat dan potensi siswa tertentu.
    6. SIMAK UI: Seleksi masuk mandiri UI untuk kelas reguler dan paralel.
    7. SIMAK UI Kelas Internasional: Seleksi untuk program internasional.
    8. SIMAK RPL: Rekognisi Pembelajaran Lampau untuk mahasiswa berpengalaman kerja atau studi sebelumnya.
